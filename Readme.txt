# Focus – Pomodoro Timer

🧠 **Focus** is a sleek, minimal, and fully responsive Pomodoro timer that helps you stay productive and manage work/break sessions efficiently.

---

## 🔹 Demo
Try it locally: [http://127.0.0.1:5500/pomodoro.html](http://127.0.0.1:5500/pomodoro.html)

---

## 🔹 How It Works

1. **Work & Break Sessions**  
   - Default work session: 25 minutes  
   - Default break session: 5 minutes  
   - Timer switches automatically between work and break sessions.

2. **Controls**  
   - **Start ▶**: Begin the timer  
   - **Pause ⏸**: Pause the timer  
   - **Reset ↻**: Reset the current session

3. **Progress & Stats**  
   - Real-time progress bar showing session completion  
   - Track total sessions, focus hours, and streaks

4. **Settings**  
   - Customize **work** and **break** durations  
   - Save settings for the next session
